About the program

Overview

This application is designed to work with BMP files, offering users a variety of functionalities, including creating new BMP files based on existing ones using different construction algorithms and color schemes, as well as embedding and extracting hidden text messages. Additionally, the program provides a user-friendly interface with the ability to save recent settings and files.

Key Features

Visual Interface:

The application has an intuitive visual interface, ensuring easy navigation and use of all available functions.

Selecting and Creating BMP Files:

  • Users can select an existing BMP file through a system file selection dialog.
  • Based on the selected file, users can create a new BMP file using one of three available construction algorithms and three color schemes.
  • The new BMP files can be saved through a system save file dialog, allowing users to choose the location and name for the new file.

Principles of Image Construction:

  • Spiral Drawing: This method calculates the color of each pixel based on its distance from the origin and its angle. Different color combinations are generated by modulating these values.
    • Combination 1: Distance, Angle, and the sum of Distance and Angle modulo 256.
    • Combination 2: Angle, the sum of Distance and Angle modulo 256, and Distance.
    • Combination 3: The sum of Distance and Angle modulo 256, Distance, and Angle.
  • Diamond Drawing(Astroid): This method generates a diamond shape centered in the image. The color is determined based on the pixel's position relative to the diamond's shape.
    • Combination 1: Red for inside the diamond, white otherwise.
    • Combination 2: Green for inside the diamond, white otherwise.
    • Combination 3: Blue for inside the diamond, white otherwise.
  • Rose Drawing: This method uses a mathematical rose curve to determine the color of each pixel. The color is based on the value of the rose curve at that point.
    • Combination 1: The red channel is the normalized value of the rose curve, the green channel is half of this value, and the blue channel is a third of this value. This creates a gradient effect with a dominance of red, followed by green and blue.
    • Combination 2: The green channel is the normalized value of the rose curve, the red channel is half of this value, and the blue channel is a third of this value. This shifts the color dominance to green, followed by red and blue.
    • Combination 3: The blue channel is the normalized value of the rose curve, the green channel is half of this value, and the red channel is a third of this value. This shifts the color dominance to blue, followed by green and red.

Each color scheme provides a unique way to visualize the patterns generated by the construction algorithms, offering a variety of visual effects for the created BMP files.

Message Embedding:

Users can "invisibly" embed a short text message into a BMP file by breaking down the byte information into bits and writing them into the least significant bits of the BMP file's color map bytes.

Message Extraction:

The application allows for the extraction of hidden text messages from BMP files, reconstructing the original text from the bits written into the file.

Authorization:

The application supports user authorization with a username and password. Upon successful authorization, the last settings and files used are saved and restored.

Activity History:

For authorized users, the application saves the last three:

  • BMP files worked on;
  • Modes of creating new bitmaps;
  • Messages embedded in BMP files;
  • Messages extracted from BMP files.

Automatic Restoration:

After authorization, users can continue working with the last settings or choose from one of the last three modes/files/messages used.

Deauthorization:

Users can deauthorize at any time while retaining the current program settings.
