User Manual

This application is designed to provide a comprehensive suite of tools for working with BMP files. Here, you'll find all the information you need to get started and make the most of the application's features.

1. Getting Started

To begin using the application, start with LogIn.

2. Toolbar

The toolbar is designed for ease of use and consists of the following sections:

  • About: Displays information about author and program
  • Crypt: Access different functions such as embedding messages and extract a message.
  • History: Displays history
  • Create BMP: Create a beautiful combination

3. Selecting and Creating BMP Files

Selecting a BMP File

  • Click on File in the menu bar.
  • Select Open to bring up the system file dialog.
  • Navigate to the desired BMP file and click Open.

Creating a New BMP File

  • After opening an existing BMP file, go to File > Create New.
  • Choose one of the three available construction algorithms and one of the three color schemes from the provided options.
  • Click Create to generate a new BMP file based on the selected parameters.
  • Save the new file by selecting File > Save As and choosing a location and filename.

4. Embedding Hidden Messages

Steps to Embed a Message

  • Click on Crypt in the menu bar and select Hide message.
  • Open the BMP file you wish to embed a message into by selecting File > Open.
  • Enter the message you want to embed in the provided text box.
  • Click Hide message to insert the message into the BMP file.
  • Save the modified BMP file by pressing Download.

5. Extracting Hidden Messages

Steps to Extract a Message

  • Click on Crypt in the menu bar and select Extract Message.
  • Open the BMP file that contains a hidden message by selecting File > Open.
  • The hidden message will be displayed in a text box.

6. User Authorization

Logging In

  • Click on Log In in the menu bar
  • Enter your username and password in the provided fields.
  • Click Log In to access your saved settings and files.

Creating an Account

  • Click on Register in the menu bar.
  • Fill in the required details and click Create.

7. Managing Activity History

For authorized users, the application keeps track of:

  • The last three BMP files you worked on.
  • The modes of creating new bitmaps you used.
  • The messages you embedded or extracted.

Accessing History

  • Click on User in the menu bar.
  • Select History to view your recent activity.
  • Choose an item from the list to load it.

8. Deauthorization

To log out from the application:

  • Click on Log Out in the menu bar.
  • Your current settings will be saved, and you can log back in to resume your work.

9. Additional Information

  • User Manual: For detailed explanations of all features, go to About > User Manual.
  • About the Program: To learn more about the application, its capabilities, and purpose, go to About > About the Program.
  • About the Authors: For information about the developers of this application, go to About > About the Authors.
